# 🌐 Personal Website Project  

This repository contains my HTML and CSS based website project.  
It includes multiple web pages and mini-projects created for practice and learning purposes.

---

## 📌 About the Project  

This project is developed using:

- HTML5 – for webpage structure  
- CSS3 – for styling and design  

The website contains different sections like:

- 📄 Personal Details Page  
- 🕒 Timetable Page  
- 🏦 Bank Management Form  
- 📝 Form Design Pages  

This project helped me understand webpage structure, styling, and organizing multiple HTML files in one project.
